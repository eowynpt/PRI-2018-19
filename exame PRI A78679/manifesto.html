Resumo:

Quanto à resolução do primeiro exercício (Vamos ao cinema) encontra-se dentro da parta cinema.
A conversão de JSON para Mongo foi feita recorrendo ao ficheiro importCinema.sh, disponível dentro da pasta cinema/mongo. Este script inseriu a informação do movies.json na base de dados cinema numa coleção a que deu o nome de movies.
Para correr a aplicação:

  • converte-se o json para mongo correndo o script importCinema.sh, disponível dentro da pasta cinema/mongo
  • corre-se num terminal o script mongod.sh disponível em cinema/mongo
  • noutro terminal faz-se npm install seguido de npm start.

Todos os 6 GETS são suportados.

A resolução do segundo exercício (CLAV) consiste de um ficheiro aquecimento.txt com a resposta às alíneas do aquecimento, e de uma pasta CLAV com a aplicação que dá resposta às últimas alíneas.
A aplicação tem a homepage em http://localhost:4008/, e responde a todas as alíneas pedidas.
